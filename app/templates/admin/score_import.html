Upload Score CSV

CSV Format — one row per player (4 rows per matchup):
matchup_id round_date course_name tee_name player_first player_last h1 … h9
  • matchup_id — integer from the reference table below
  • round_date — YYYY-MM-DD (blank = uses scheduled date)
  • course_name / tee_name — partial match OK; blank = uses matchup's assigned course/tee
  • player_first / player_last — must match a player on that matchup's teams
  • h1–h9 — integer gross score per hole (1–20)
  • Each matchup needs exactly 4 player rows. Already-completed matchups are skipped.
