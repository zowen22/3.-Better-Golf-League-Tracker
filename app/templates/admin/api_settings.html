Quick Start

Authenticate by adding your key to any request:

  -H "X-Api-Key: {{ api_key or 'YOUR_API_KEY' }}"

Or via query string:

{{ request.host_url }}api/v1/seasons?api_key={{ api_key or 'YOUR_API_KEY' }}
