# Plans

Side-by-side comparisons of two or more meaningful implementation approaches,
used when the right path isn't obvious enough to just pick and go.

Lifecycle: `Evaluating` → `Decision: [chosen approach]` → `Archived`

Required fields per plan: Approach A / Approach B (each with summary,
tradeoffs, effort estimate, decision criteria).

When resolved: record the chosen approach in `2. Project Overview.md`'s
Decisions Log, then archive the plan file (leave it in place with
`Status: Archived` — it's the historical record of why).
