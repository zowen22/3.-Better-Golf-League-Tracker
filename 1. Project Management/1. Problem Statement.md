# Problem Statement

*Source: [x] Handoff document — `Handoff.txt` (May 26, 2026)*

-----

## Problem

Golf leagues lack a good self-hosted management option. The dominant tool (golfleaguetracker.com / GLT) is SaaS-only with no self-hosted path, limited customization, and subscription cost. League admins managing small recreational leagues need something they can own and run themselves.

## Target User

- **Primary:** Golf league administrators — set up seasons, manage schedules, enter scores, run the league week to week
- **Secondary:** League members — view standings, scorecards, stats, forum
- **Future:** Any recreational golf league willing to pay for a hosted version

## Goals

1. Full-featured replacement for GLT, self-hosted on personal infrastructure (NAS → Cloudflare Tunnel)
2. Feature parity with GLT (achieved as of Jun 2026) plus some improvements
3. Long-term: pivot to paid SaaS platform — each league is a tenant, Stripe billing

## Core Features

- Multi-tenant architecture (each league isolated)
- Match play scoring with handicaps (par-based differential)
- Season / team / schedule management
- Standings (5 views), skins, playoffs, records, stats
- Admin panel with 250+ configurable settings
- Individual user accounts + shared league password (dual auth)
- Self-reporting, sub requests, email notifications, forum

## Constraints

- Self-hosted first — must run on a home PC / NAS (now: Render + Supabase for demo/prod)
- No USGA API integration (par-based handicap by design, lower complexity)
- No native mobile apps (mobile-optimized web only)
- No SQLAlchemy ORM — raw SQL throughout (simplicity, control)
